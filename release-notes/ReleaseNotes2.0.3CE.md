Please note that Magento Open Source Edition 2.0.3 has been replaced by Magento 2.0.4. This release includes all of the security enhancements and performance improvements of Magento 2.0.3, in improved packaging. **You must download and install 2.0.4 to ensure that you receive all the security enhancements of 2.0.3**. Please see  [Magento Open Source Release Notes 2.0.4]({{ page.baseurl }}/release-notes/ReleaseNotes2.0.4CE.html){: target="_blank"} for download information.
