## Functions

The following functions unsubscribe from events.

### mse.unsubscribe.addToCart

Unsubscribes from the `addToCart` event.

|Name|Required|Description|
|---|---|---|
|`handler`|Yes|Event handler|

Syntax:

```javascript
mse.unsubscribe.addToCart(handler);
```
