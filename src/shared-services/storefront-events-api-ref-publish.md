## Functions

The following functions publish events to notify all subscribers.

### mse.publish.addToCart

Publishes the `addToCart` event.

|Name|Required|Description|
|---|---|---|
|`context`|No|Custom context|

Syntax:

```javascript
mse.publish.addToCart(ctx);
```
