Catalog module

POST V1/categories

When you create a category from Admin, you have the option to configure display and search engine optimization settings. To configure these settings using REST, you can specify the following parameters as attribute_code values:

all_children
children
children_count
custom_apply_to_products
custom_design
custom_design_from
custom_design_to
custom_layout_update
custom_use_parent_settings
default_sort_by
description
display_mode
filter_price_range
image
is_anchor
landing_page
meta_description
meta_keywords
meta_title
page_layout
path
path_in_store
url_key
url_path

Third-party modules may define other custom attributes.
