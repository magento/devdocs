<h2 id="compman-overview">Overview of the Component Manager</h2>
This section discusses how to start the Component Manager, which updates or uninstalls Magento components:

*	Modules (extend Magento capabilities)
*	Themes (change the look and feel of your storefront and Admin)
*	Language packages (localize the storefront and Admin)

<div class="bs-callout bs-callout-warning">
    <p>If you installed the Magento application by <a href="{{ site.gdeurl21 }}install-gde/prereq/dev_install.html">cloning the GitHub repository</a>, you <em>cannot</em> use the Component Manager utility to update components. Instead, you must <a href="{{ site.gdeurl21 }}install-gde/install/cli/dev_options.html">update them manually</a>.</p>
</div>
