Magento Enterprise B2B Extension configuration paths reference

This topic lists configuration paths for Magento Enterprise B2B Extension. The magento app:config:dump command writes these values to the shared configuration file, app/etc/config.php, which should be in source control.

This reference lists only configuration paths unique to Magento Enterprise B2B Extension. Magento Enterprise B2B Extension includes all configuration paths for Magento Open Source and Magento Commerce as well.

For those configuration paths, see:

  • Payment configuration paths
  • Sensitive and system-specific configuration paths reference

To optionally override any configuration settings or to set sensitive settings, see Use environment variables to override configuration settings.

This reference contains unique configuration paths for the Magento Enterprise B2B Extension. The Magento Enterprise B2B Extension supports all configuration paths for Magento Open Source (formerly Community Edition (CE) and Magento Commerce (formerly Enterprise Edition (EE) as well.
